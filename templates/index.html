Instructions : How to use the Model

  • Steps:These are the steps to use this model

    • Upload the file: You can upload the file by clicking on the hoose file button.

    • Predicted file:Now you can download the predictions as a CSV file.

    These steps will help ensure a successful implementation of the Sensor Fault Detection model. If you have any questions or need assistance, feel free to reach out!
