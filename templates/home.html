About the ML Model

Welcome to the Sensor Fault Detection Project! This project aims to develop a machine learning model that classifies semiconductor wafers as either "Good" or "Bad" based on sensor readings. To successfully navigate through this project, please follow these instructions:

  • Data Collection: The Dataset contains sensor readings for the wafers. This dataset is crucial for training and testing the machine learning model.
